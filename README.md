## 👨‍💻 Sobre mí

Estudiante de Licenciatura en Sistemas en la **Universidad ORT Uruguay** y founder de **Travel Plan** — una app de planificación de viajes grupales disponible en el App Store.

Me apasiona construir productos reales: desde la arquitectura backend hasta la experiencia de usuario final.

## 🚀 Proyecto destacado

### [Travel Plan — App de viajes grupales](https://travel-wishlist.com)

> App móvil fullstack para planificar viajes solos o en grupo, con itinerarios compartidos, gastos, checklists, votaciones y más. Publicada en el App Store.

**Stack:** React · TypeScript · Capacitor · Supabase · Tailwind CSS · Vite  
**Features:** Itinerarios · Gastos grupales · Votaciones · Tiempo real · Exportación PDF
